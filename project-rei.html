Rei-kun Bot

The Persona-Driven AI Orchestrator

Core Architecture

Rei-kun is a production-grade Discord orchestrator serving an active student community. Unlike basic bots, it utilizes a multi-model fallback engine via OpenRouter, ensuring 100% uptime by switching between DeepSeek, LLaMA 4, GPT, and Gemini.

Key technical features include:

  • Code Doctor: A proprietary tool that scans Python source code, flags issues, and proposes patches with rollback support.
  • Resource Hub: Interactive wizard allowing students to retrieve notes, mind maps, and quizzes via code.
  • Dynamic Memory: Per-channel conversation memory for a persistent persona.
  • Hot-Reload: Architecture that allows module updates without restarting the bot.
