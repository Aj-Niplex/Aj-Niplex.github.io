Niplex-MCP

Secure Model Context Protocol Server

Overview

A secure MCP server designed to bridge the gap between AI agents and private codebases. It allows agents to interact with GitHub repositories without exposing sensitive Personal Access Tokens (PATs) to the model itself.

By utilizing a proxy-based architecture, Niplex-MCP ensures that the AI agent receives the necessary context while the credentials remain securely vaulted on the server.

Pull & Run
git clone https://github.com/Aj-Niplex/niplex-mcp.git
cd niplex-mcp
pip install -r requirements.txt
# Run the MCP server
python server.py
