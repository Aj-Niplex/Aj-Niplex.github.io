Niplex-MCP

Secure Model Context Protocol Server

The Secure Bridge

Niplex-MCP is a Model Context Protocol (MCP) server that acts as a high-security bridge between AI tools (like Claude) and private GitHub repositories.

The core problem it solves is credential exposure. Instead of giving an AI agent a raw Personal Access Token (PAT), Niplex-MCP handles the authentication on the server side, allowing the agent to read code and list files without ever seeing the secret keys.

Deployment
git clone https://github.com/Aj-Niplex/niplex-mcp.git
cd niplex-mcp
pip install -r requirements.txt
python server.py
